# 窗帘尺寸测量指南

## 测量工具准备

在测量窗帘尺寸之前，请准备以下工具：

- **钢卷尺**（5米以上）：首选工具，精度高不易变形，避免使用布尺（会拉伸导致误差）
- **铅笔和纸**：记录每个窗户的尺寸数据，建议画简图标注
- **梯子/凳子**：用于测量高处尺寸，确保安全稳固
- **水平仪**（可选）：检查窗框是否水平，判断是否需要修正尺寸
- **手机**：拍照记录窗户现场情况（暖气片、空调位置等）

测量建议：每个尺寸至少测量3个点（左、中、右或上、中、下），取最大值作为最终数据。

## 窗户宽度测量方法

### 外装（罗马杆安装）

外装是指窗帘杆安装在窗框外侧墙面上，窗帘覆盖范围大于窗框。

**测量步骤：**
1. 测量窗框外沿的总宽度（A）
2. 两侧各加15-20厘米的延伸量（遮挡窗框两侧漏光）
3. 最终窗帘杆长度 = A + 30~40厘米
4. 确认两侧墙面空间足够（无柜子、墙角等阻挡）

**示例：** 窗框外宽150厘米 → 窗帘杆长度建议190厘米（两侧各延伸20厘米）

**注意事项：**
- 如果两个窗户距离较近（间距小于30厘米），建议合并使用一根长杆
- 窗帘杆两端距墙壁至少保留5厘米空间（装饰头所需）
- 罗马杆长度不宜超过3.5米，否则中间会下垂，需加中间支架

### 内装（轨道嵌入安装）

内装是指窗帘轨道安装在窗帘盒内部或窗框内侧。

**测量步骤：**
1. 测量窗帘盒内侧净宽度（B）
2. 轨道长度 = B - 1~2厘米（预留安装间隙）
3. 如果是窗框内安装，测量窗框内宽，轨道长度 = 内宽 - 1厘米

**注意事项：**
- 双轨安装时，窗帘盒深度需≥15厘米（前后两条轨道各需5-7厘米）
- 确认窗帘盒内是否有预埋木方（用于固定轨道螺丝）
- 弯轨安装需测量窗帘盒的弯折角度和弯折半径

## 窗户高度测量方法

### 到窗台（半截窗帘）

适用场景：窗台下方有暖气片、矮柜等家具。

**测量步骤：**
1. 从窗帘杆/轨道安装位置向下量至窗台面
2. 再向下延伸15-20厘米（遮挡窗台下方）
3. 最终窗帘高度 = 杆到窗台距离 + 15~20厘米

**注意事项：**
- 如果窗台下有暖气片，窗帘不要盖住暖气片（影响散热），底部应在暖气片上方5厘米
- 窗帘底部与窗台下的家具保持2-3厘米间距

### 到地面（落地窗帘）

适用场景：客厅、卧室等主要空间，视觉效果最佳。

**测量步骤：**
1. 从窗帘杆/轨道安装位置向下量至地面
2. 减去2-3厘米（防止拖地积灰）
3. 最终窗帘高度 = 杆到地面距离 - 2~3厘米

**特殊情况处理：**
- 地面不平（如有踢脚线区域高低差）：取最短距离，确保窗帘不拖地
- 安装地暖的房间：考虑地面铺完地板后的最终高度
- 拖地款窗帘：高度 = 杆到地面距离 + 5~10厘米（刻意多出堆叠在地面）

### 落地窗/阳台门

**测量步骤：**
1. 从安装位置量至地面
2. 减去1-2厘米
3. 注意门框上方到吊顶的距离是否足够安装杆/轨

**注意事项：**
- 阳台推拉门需确认窗帘不影响门的开合
- 如有门槛石，以门槛石顶面为基准测量

## 罗马杆安装的尺寸计算

### 安装位置确定

- **高度**：窗框上沿往上10-15厘米安装支架（提升视觉高度），如果距吊顶较近（＜15厘米），建议贴近吊顶安装
- **宽度**：窗框外沿两侧各延伸15-20厘米
- **离墙距离**：支架前端距墙面10-15厘米（确保窗帘不贴窗框）

### 尺寸计算公式

| 项目 | 计算方式 |
|------|----------|
| 杆长度 | 窗框外宽 + 30~40厘米 |
| 支架数量 | 杆长≤2米用2个支架，2-3米用3个支架 |
| 窗帘宽度 | 杆长度 × 褶皱倍数 |
| 窗帘高度 | 杆中心到目标位置 - 挂环高度（约5-8厘米）|
| 挂环数量 | 窗帘宽度(米) ÷ 0.1 = 挂环个数（约每10厘米一个）|

### 罗马杆直径选择

| 杆长度 | 推荐直径 | 承重能力 |
|--------|----------|----------|
| ≤2.0米 | 28mm | 约10-15kg |
| 2.0-3.0米 | 32mm | 约15-20kg |
| 3.0-3.5米 | 35mm | 约20-25kg（需中间支架）|

## 轨道安装的尺寸计算

### 安装位置确定

- **窗帘盒内安装**：轨道固定在窗帘盒顶部木方上
- **无窗帘盒/顶装**：轨道直接固定在天花板上，距墙面10-15厘米
- **侧装**：轨道支架固定在墙面上（类似罗马杆位置）

### 尺寸计算公式

| 项目 | 计算方式 |
|------|----------|
| 轨道长度（窗帘盒内） | 窗帘盒内净宽 - 1~2厘米 |
| 轨道长度（顶装/侧装） | 窗框外宽 + 30~40厘米 |
| 双轨间距 | 前后轨道间隔5-8厘米 |
| 窗帘宽度 | 轨道长度 × 褶皱倍数 |
| 窗帘高度 | 轨道底部到目标位置 - 挂钩露出长度（约2-4厘米）|

### 弯轨尺寸计算（飘窗/弧形窗）

- 直线段长度：测量每段直线的实际长度
- 弯折段：测量弯折角度（通常90°或120°），弯折半径建议≥10厘米
- 总长度 = 所有直线段长度之和 + 弯折段弧长
- U型飘窗轨道总长 = 正面宽度 + 左侧深度 + 右侧深度 + 弯折余量（每个弯约加5厘米）

## 褶皱倍数与面料用量计算

### 褶皱倍数对比

| 褶皱倍数 | 效果描述 | 适用面料 | 适用场景 |
|----------|----------|----------|----------|
| 1.5倍 | 褶皱较浅，略显平整 | 纱帘、薄面料 | 预算有限、纱帘 |
| 2.0倍 | 褶皱自然美观（推荐） | 通用 | 大多数家庭场景 |
| 2.5倍 | 褶皱丰满饱满 | 丝绒、雪尼尔等厚面料 | 轻奢/欧式风格 |
| 3.0倍 | 褶皱极度饱满 | 薄纱 | 高端定制、婚礼布置 |

### 面料用量计算公式

**单开窗帘（一片）：**
面料宽度 = 窗帘杆/轨道长度 × 褶皱倍数 + 两侧折边余量（共10-20厘米）

**对开窗帘（两片）：**
每片面料宽度 =（窗帘杆/轨道长度 ÷ 2）× 褶皱倍数 + 中间重叠量5厘米 + 侧边折边余量5-10厘米

**面料高度用量：**
面料高度 = 成品窗帘高度 + 上折边（10厘米）+ 下折边（15-20厘米）

### 计算示例

**场景：** 窗户宽2.5米，需要落地窗帘，罗马杆安装，选择2倍褶皱。

- 窗帘杆长度：2.5 + 0.4 = 2.9米
- 窗帘面料总宽：2.9 × 2.0 + 0.2 = 6.0米
- 对开每片宽度：6.0 ÷ 2 = 3.0米
- 窗帘成品高度（假设杆到地面2.8米）：2.8 - 0.08（挂环）- 0.02（离地）= 2.7米
- 面料高度用量：2.7 + 0.10 + 0.15 = 2.95米
- 面料门幅1.4米，每片需拼幅：3.0 ÷ 1.4 ≈ 3幅（向上取整）
- 总面料米数：3幅 × 2片 × 2.95米 = 17.7米

## 飘窗尺寸测量

### 飘窗内安装（台面上方安装轨道）

**需要测量的尺寸：**

1. **正面宽度（A）**：飘窗正面玻璃框的内宽
2. **左侧深度（B）**：从正面到左侧墙面的深度
3. **右侧深度（C）**：从正面到右侧墙面的深度
4. **高度（D）**：飘窗顶部到台面的净高度
5. **顶部宽度（E）**：飘窗顶部的宽度（确认是否可以安装轨道）

**轨道尺寸计算：**
- U型弯轨总长 = A + B + C + 弯折余量（约10厘米）
- 窗帘高度 = D - 1厘米（离台面1厘米）
- 窗帘面料总宽 =（A + B + C）× 褶皱倍数

### 飘窗外安装（落地窗帘）

**需要测量的尺寸：**
1. 飘窗外侧墙面的总宽度
2. 安装位置到地面的高度
3. 飘窗台面凸出墙面的深度（决定窗帘杆离墙距离）

**注意：** 窗帘杆/轨道离墙距离必须大于飘窗台面凸出深度+5厘米，否则窗帘会被台面挡住。

## 弧形窗/异形窗处理

### 弧形窗（弧形飘窗、弧形阳台）

**测量要点：**
1. 用软尺沿弧形窗框外沿测量总弧长
2. 测量弧形的弦长（两端直线距离）和拱高（弧形最高点到弦的距离）
3. 将弧形分为多段直线近似测量（每段30-50厘米），记录转折角度
4. 高度测量同普通窗户

**安装方案：** 必须使用可弯曲轨道（铝合金弯轨），由专业师傅现场根据弧度弯折定型。弯轨价格约为直轨的1.5-2倍。

### 三角形窗/斜窗

**测量要点：**
1. 测量三角形/梯形的各边长度
2. 测量斜边的倾斜角度
3. 测量最高点和最低点的高度

**安装方案：** 可选斜拉式百叶帘、定制异形卷帘，或固定式装饰窗帘（不开合）。

### 拱形窗

**测量要点：**
1. 测量窗户底边宽度
2. 测量直线段高度
3. 测量拱形段的半径和弧度

**安装方案：** 直线段部分安装普通窗帘，拱形段使用定制扇形帘或窗贴膜。

## 幔头尺寸计算

### 常见幔头类型与尺寸

| 幔头类型 | 高度建议 | 宽度计算 | 面料用量 |
|----------|----------|----------|----------|
| 平幔 | 窗帘总高的1/5-1/6（约30-50厘米）| 与窗帘杆等宽 | 宽度×1.5-2倍 |
| 水波幔 | 30-60厘米 | 与窗帘杆等宽 | 宽度×2.5-3倍 |
| 工字幔 | 20-40厘米 | 与窗帘杆等宽 | 宽度×2倍 |

### 幔头安装要求

- 需要额外一条轨道或幔头专用支架
- 双轨+幔头 = 三轨安装，窗帘盒深度需≥20厘米
- 罗马杆安装通常不搭配幔头（罗马杆本身是装饰元素）
- 幔头下沿不应遮挡窗户上沿（保证采光）

## 常见测量错误和注意事项

### 常见错误

1. **只测量一个点**：窗框可能不完全方正，上下宽度可能有1-3厘米差异，必须多点测量取最大值
2. **忘记预留安装空间**：罗马杆两端装饰头各需5-8厘米空间，轨道堵头各需1-2厘米
3. **忽略障碍物**：窗户周围的空调内机、暖气片、开关面板、插座等会影响窗帘安装位置
4. **混淆内装和外装尺寸**：内装量窗框内宽，外装量窗框外宽+延伸量，两者差别很大
5. **地面不平未考虑**：部分房间地面有坡度或高低差，需在多个位置测量到地面距离
6. **忘记扣除挂钩/挂环高度**：挂环式约减5-8厘米，挂钩式约减2-4厘米
7. **门窗开合冲突**：未考虑窗帘是否会影响窗户开合、阳台门推拉
8. **飘窗台面凸出未测量**：导致落地窗帘被台面挡住

### 注意事项清单

- 测量前确认装修是否全部完成（地面、吊顶、窗帘盒）
- 确认电动窗帘的电源插座位置是否合适
- 确认窗帘盒内是否预埋了木方（用于固定轨道螺丝）
- 多窗户房间注意保持窗帘底边在同一水平线上
- 高层住宅注意窗户附近是否有强风，选择适当重量的面料
