# AI 客服系统 UI 设计规范

> 版本：v8.0  
> 日期：2026-04-12  
> 适用范围：C 端客服对话页 / 客服员工工作台 / 企业管理后台

---

## 一、设计语言

### 1.1 设计理念

**关键词**：专业、清晰、温暖、布艺质感

系统面向布艺行业（窗帘生产与家装）企业，UI 风格应体现：
- **专业感**：管理后台以数据驱动，信息密度高但不拥挤
- **温暖感**：C 端对话页体现亲切、可信赖的客服体验
- **布艺质感**：色彩柔和、圆角适度、阴影轻盈，避免生硬的工业感

### 1.3 字体系统

#### 字族

```css
font-family: -apple-system, BlinkMacSystemFont, "SF Pro SC", "SF Pro Text",
  "SF Pro Icons", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei",
  "Helvetica Neue", Helvetica, Arial, sans-serif;
```

**中文优先**：PingFang SC（macOS）/ Microsoft YaHei（Windows）作为中文主字体。

#### 字号阶梯

| 级别 | 字号 | 行高 | 字重 | 用途 |
|------|------|------|------|------|
| H1 | 24px | 32px | 600 | 页面标题 |
| H2 | 20px | 28px | 600 | 区块标题 |
| H3 | 16px | 24px | 600 | 卡片标题、Modal 标题 |
| Body-L | 15px | 24px | 400 | 管理后台正文 |
| Body | 14px | 22px | 400 | **默认正文** |
| Body-S | 13px | 20px | 400 | 表格内容、表单文字 |
| Caption | 12px | 18px | 400 | 辅助说明、时间戳 |
| Caption-XS | 11px | 16px | 400 | 极小标签、角标数字 |

#### 数字字体

金额、库存、销量等关键数字使用 `tabular-nums` 等宽数字：

```css
font-variant-numeric: tabular-nums;
```

## 二、组件设计规范

### 2.1 按钮

| 类型 | 背景 | 文字 | 边框 | 场景 |
|------|------|------|------|------|
| Primary | `#2F54EB` | `#FFF` | 无 | 主操作（搜索、提交并上架、发货）|
| Default | `#FFF` | `#595959` | `#D9D9D9` | 次要操作（重置、存草稿）|
| Text | 透明 | `#2F54EB` | 无 | 行内操作（查看、编辑、下架）|
| Danger | `#FFF` | `#FF4D4F` | `#FF4D4F` | 危险操作（删除、关闭订单）|
| Link | 透明 | `#2F54EB` | 无 | 纯文字链接 |

**按钮尺寸**：

| 尺寸 | 高度 | 内边距 | 字号 | 用途 |
|------|------|--------|------|------|
| Large | 40px | 0 24px | 15px | 主操作按钮 |
| Default | 32px | 0 16px | 14px | 表单操作、列表操作 |
| Small | 24px | 0 12px | 13px | 表格操作列 |

### 2.2 输入框

```
默认态：边框 #D9D9D9，背景 #FFF
悬浮态：边框 #597EF7
聚焦态：边框 #2F54EB，外发光 0 0 0 2px rgba(47,84,235,0.15)
错误态：边框 #FF4D4F，外发光 0 0 0 2px rgba(255,77,79,0.15)
禁用态：边框 #D9D9D9，背景 #F5F5F5，文字 #BFBFBF
```

**高度**：32px（Default）/ 24px（Small）  
**圆角**：4px  
**占位文字颜色**：`#BFBFBF`

### 2.3 表格

| 元素 | 样式 |
|------|------|
| 表头 | 背景 `#FAFAFA`，文字 `#595959` 600 字重，高度 48px |
| 行 | 高度 56px（默认）/ 72px（含多行文字的商品行）|
| 行悬浮 | 背景 `#F0F5FF` |
| 行选中 | 背景 `#E6F0FF`，左侧 3px `#2F54EB` 边框 |
| 分割线 | `#E8E8E8` 1px |
| 排序图标 | 表头右侧，未排序 `#BFBFBF`，激活 `#2F54EB` |

### 2.5 消息气泡（C 端对话页）

**AI 消息**：
- 背景：`#FFF`
- 阴影：Shadow-4（轻盈上浮感）
- 圆角：16px（左上 16px、右上 16px、右下 4px、左下 16px）
- 最大宽度：70% 屏幕宽度
- 左侧显示 AI 头像（圆形 32px，主色渐变背景 + 小布文字）

**用户消息**：
- 背景：`#2F54EB`
- 文字：`#FFF`
- 圆角：16px（左上 16px、右上 16px、左下 4px、右下 16px）
- 最大宽度：70% 屏幕宽度
- 右对齐

### 3.2 欢迎区设计

首次访问时显示欢迎区，背景使用柔和渐变色：

```
background: linear-gradient(135deg, #F0F5FF 0%, #FFF7E6 100%);
```

四个快捷功能入口为卡片式按钮，2×2 网格布局：

```
┌──────────────────┬──────────────────┐
│  📦 查订单       │  🛍️ 找产品       │
│  14px, #595959   │  14px, #595959   │
├──────────────────┼──────────────────┤
│  🔄 退换货       │  📞 转人工       │
│  14px, #595959   │  14px, #595959   │
└──────────────────┴──────────────────┘
```

每个按钮：背景 `#FFF`，圆角 12px，Shadow-1，hover 时上浮 2px + Shadow-2。

### 3.3 打字机效果

AI 回复时使用打字机效果：
- 逐字出现，间隔 20ms/字
- 光标闪烁动画（`|`，宽度 2px，颜色 `#2F54EB`）
- 流式输出时不等待完整消息，收到 SSE `text` 事件即追加

### 3.4 订单卡片样式

AI 返回订单信息时使用卡片组件：

## 六、响应式适配

### 6.1 断点

| 断点 | 宽度 | 适配策略 |
|------|------|---------|
| XS | < 576px | C 端小程序全屏对话 |

| XL | > 1440px | 管理后台宽屏，表格列数增加 |

| 宽度 | 侧边栏 | 主内容区 |
|------|--------|---------|

| 768px - 1024px | 收起 64px（仅图标）| 正常 |
| < 768px | 隐藏（抽屉式，点击汉堡菜单展开）| 全屏 |

## 七、动画与微交互

### 7.1 过渡动画

| 场景 | 动画 | 时长 |
|------|------|------|
| 页面切换 | Fade in + slide up | 200ms ease-out |
| Modal 打开 | Scale 0.9 → 1 + fade in | 200ms ease-out |
| Drawer 打开 | Slide in from right | 250ms ease-out |
| 下拉菜单 | Fade in + slide down | 150ms ease-out |
| Toast 提示 | Slide in from top + fade in | 200ms ease-out |

### 7.2 微交互

| 元素 | 交互 |
|------|------|
| 按钮 Hover | 背景色深 10%，上浮 1px |
| 表格行 Hover | 背景变为 `#F0F5FF` |
| 卡片 Hover | 阴影加深（Shadow-1 → Shadow-2），上浮 2px |
| 库存预警数字 | 脉冲动画（opacity 1 → 0.6 → 1，1.5s 循环）|
| 在线客服状态 | 绿色圆点呼吸动画（scale 1 → 1.2 → 1，2s 循环）|
| 打字机光标 | 闪烁动画（opacity 1 → 0 → 1，0.8s 循环）|
