# Data Skill — Few-shot 示例

## ✅ 正确示例

### 例1: 查看今日经营概览
用户: "今天生意怎么样"
```
→ dashboard_stats(action="overview")
→ 📊 今日经营概览：
  • 订单数：12单
  • 销售额：¥3,240
  • 客单价：¥270
  • 活跃客服：3人在线
```

### 例2: 查看订单趋势
用户: "最近一周订单情况"
```
→ dashboard_stats(action="order_trend", days=7)
→ 近7天订单趋势：
  06-06: 8单 · 06-07: 15单 · 06-08: 10单 · 06-09: 12单
  06-10: 18单 · 06-11: 14单 · 06-12: 12单
→ "周末订单量较高，06-10达到峰值18单。整体趋势平稳。"
```

### 例3: 查看客服会话
用户: "现在有多少人在排队"
```
→ session_manage(action="monitor")
→ 客服监控：
  • 在线客服：3人
  • 活跃会话：5个
  • 排队人数：2人
  • 平均等待：30秒
```

## ❌ 错误示例

### 反例1: 编造经营数据
用户: "这个月销售额多少"
```
❌ "大约10万左右"（未调工具，编造数据）
✅ 调 dashboard_stats(action="overview") → 展示真实数据
```

### 反例2: 数据展示不标来源
用户: "今天订单状态分布"
```
❌ "已完成8单，待发货4单"（未标来源）
✅ dashboard_stats(action="order_status") → "已完成8单，待发货4单"
```
