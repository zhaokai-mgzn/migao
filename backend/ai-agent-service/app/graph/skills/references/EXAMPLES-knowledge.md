# Knowledge Skill — Few-shot 示例

> ⚠️ RAG 知识库当前禁用。knowledge_search 工具未注册。启用 RAG 后取消此注释。

## ✅ 正确示例

### 例1: 查面料知识
用户: "高精密面料有什么特点"
```
→ knowledge_search(query="高精密面料特点")
→ 📖 高精密面料 [知识库]：
  • 采用高密度织造工艺，纱线细密
  • 手感细腻光滑，光泽度好
  • 适合现代简约、轻奢风格
  • 保养：建议干洗，避免暴晒
→ "以上信息来自知识库 [工具返回]。需要看相关商品吗？"
```

### 例2: 查加工费标准
用户: "打孔加工怎么收费"
```
→ knowledge_search(query="打孔加工费标准")
→ 💰 打孔加工费 [知识库]：
  • 常规打孔：3元/米
  • 精品打孔：5元/米（含孔环）
  • 罗马圈打孔：8元/米
  • 以上为参考价格，具体以实际订单为准
→ "数据来源于知识库 [工具返回]。需要帮您算具体的加工费吗？"
```

### 例3: 查安装指南
用户: "窗帘怎么量尺寸"
```
→ knowledge_search(query="窗帘尺寸测量方法")
→ 📐 窗帘测量指南 [知识库]：
  • 宽度：轨道长度 × 1.5~2倍（褶皱系数）
  • 高度：轨道到地面距离 - 1~2cm（离地间隙）
  • 飘窗：按窗框内径测量
→ "以上为通用测量建议 [知识库]。建议由专业师傅上门测量更准确。"
```

## ❌ 错误示例

### 反例1: 知识库没有时编造
用户: "我们家的售后政策是什么"
```
❌ "7天无理由退换，30天质保"（未核实，编造政策）
✅ knowledge_search(query="售后政策") → 如未找到："知识库暂未收录售后政策 [工具返回]，建议查阅公司内部文件或咨询主管。"
```

### 反例2: 不标知识来源
用户: "雪尼尔和棉麻哪个好"
```
❌ "雪尼尔更好"（未标来源，主观判断）
✅ knowledge_search → "雪尼尔：手感柔软，适合卧室 [知识库]；棉麻：透气自然，适合客厅 [知识库]。选择取决于使用场景，您更看重哪方面？"
```
