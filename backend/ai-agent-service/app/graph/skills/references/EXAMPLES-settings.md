# Settings Skill — Few-shot 示例

## ✅ 正确示例

### 例1: 查看系统设置
用户: "看一下当前的系统设置"
```
→ settings_manage(action="get_settings")
→ 展示当前配置 [工具返回]：
  • 商户名称：词元通达
  • 行业：窗帘布艺
  • 客服电话：400-xxx-xxxx
```

### 例2: 修改AI配置
用户: "把AI客服的问候语改成'您好，欢迎咨询词元通达！'"
```
→ 先确认："确认将AI客服问候语修改为'您好，欢迎咨询词元通达！'吗？"
→ 用户确认后: settings_manage(action="update_ai_config", greeting="您好，欢迎咨询词元通达！")
→ "AI客服问候语已更新 [工具返回]"
```

### 例3: 查看登录日志
用户: "最近谁登录过系统"
```
→ settings_manage(action="login_logs")
→ 展示最近登录记录 [工具返回]：
  • 张三 · admin · 06-12 09:30 · IP: 192.168.1.x
  • 李四 · agent · 06-12 08:15 · IP: 192.168.1.y
```

## ❌ 错误示例

### 反例1: 不确认直接修改
用户: "把系统名称改成测试商户"
```
❌ 直接调 settings_manage(action="update_settings", name="测试商户")
✅ 先确认："确认将商户名称从'词元通达'改为'测试商户'吗？此修改影响所有页面展示。"
```

### 反例2: 泄露密码相关信息
用户: "帮我看看张三的密码是什么"
```
❌ "张三的密码是xxx"（密码不应明文展示）
✅ "抱歉，密码属于敏感信息，无法查看。如需重置密码，我可以帮您操作。"
```
