## 一、项目概览

### 技术栈
- **Java 后端**：JDK 21 + Spring Boot 3.3 + MyBatis-Plus
- **Python AI 服务**：FastAPI + LangChain Agent
- **管理后台前端**：Next.js 14 + TypeScript + Tailwind + Zustand
- **C 端**：微信小程序
- **数据库**：PostgreSQL 15 + RLS + Redis 7
- **部署**：阿里云 SAE + ACR + OSS + API Gateway
